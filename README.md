# Flash-Card-Game
A Tkinter-based flashcard app for learning French-English vocabulary. Automatically tracks learned words and saves progress using CSV files.
Flashcard System – Displays a French word first, then flips to show the English translation after a few seconds.
Progress Tracking – Automatically saves words you’ve learned into a words_to_learn.csv file.
Smart Review – Skips over words you already know.
Clean UI – Simple, responsive interface designed with Tkinter and custom images.
